# Automobile_KNN_Price_Prediction
Jupyter Notebook project - Cleaning and normalizing a used car sales database, then initializing and training univariate and multivariate K-Nearest-Neighbor algorithm to predict prices. 
Working through hyperparameter optimization for uni and multivariate KNNs. 
Inferring best multivariate feature combinations and computing KNN combination of features / hyperparameters resulting in lowest RMSE.
